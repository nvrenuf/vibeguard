# Roadmap

## v0 (this repo first pass)
- CLI: run gates + emit findings + export audit pack
- Policy bundle schema + baseline bundle
- CI workflow: run `make verify` + smoke check

## v1
- GitHub PR integration (comment on PR with findings)
- Approvals as code (CODEOWNERS + required reviewers mapping)
- Policy bundle signing / provenance (SLSA-style)

## v2+
- IDE extension / UI “Wizard”
- Multi-repo org policies and exceptions
- Agent tool integrations (Codex task runner hooks)
